회식 · 퇴근 후 메시지 — 위축을 만드는 "규범 없는 영역"

해결: 금지가 아니라 명시적 규칙 + 선택권

🍻 회식 (Hoesik) 규범 설계

❌ AS-IS (암묵)

• 참석 = 의무
• 음주 강권
• 2~3차 관행
• 귀가 시 "벌써?"

✅ TO-BE (명시)

• 참석 선택권
• 음주/비음주 동등
• 1차 종료 시간 고정
• 불참 사유 불문

💡 "회식은 업무의 연장이 아니라 팀 문화의 선택지"

📱 퇴근 후 메시지 규범 설계

❌ AS-IS (암묵)

• 24시간 응답 기대
• 읽씹 = 불충
• 야간 업무 지시
• 공사 경계 붕괴

✅ TO-BE (명시)

• 응답 SLA 명시
• 야간 '확인만' 규칙
• 긴급 = 전화만
• 주말 메시지 금지

Case Clinic — 위축 사건을 명시적 규범으로 전환하기

Theory U 기반 · 사건 → 구조 분석 → 운영 프로토콜

1

사건 서술 (2분)

발표자가 구체적 위축 장면 설명 — 이름 없이, 상황만

2

상황 질문 (3분)

청중이 상황 질문만 — "그때 다른 사람은?"

3

구조 분석 (5분)

트리거 4요소 분해: 상황·행동·반응·결과

4

규범 초안 작성 (5분)

"이 장면이 반복되지 않으려면, 어떤 규칙이 필요한가?"

5

합의 선언 (2분)

참가자 전원 조건부 합의 → 팀 운영 규범에 반영

⚙️ 운영 원칙

🛡️ 비밀유지: 외부 반출 금지

⚖️ 비난 금지: 행동/조건만 분석

📝 사실 우선: 해석 후순위

⏱️ 시간 엄수: 각 단계 타이머

📋 예시: 산출물

사건

"신입이 회의 중 질문하니 팀장이 한숨을 쉬었고, 이후 아무도 질문하지 않음"

구조 진단

감정 표출(한숨) → 관찰 학습 → 질문 소멸

규범 초안

위계(Hierarchy) = 발언 억제 장치인가, 보호 장치인가?

"위계를 없앨 수 없다면, 위계의 의미를 바꿔라" — Hierarchy as Responsibility

❌ 위계 = 특권

• 상사 말 = 정답

• 경력 = 발언권

• 반대 = 도전

• 실수 = 무능의 증거

⬇️

아래에서 위로 말 못함

✅ 위계 = 책임

• 상사 역할 = 결정 품질 보장

• 경력 = 질문 의무

• 반대 = 팀 보호 행위

• 실수 = 학습 기회

⬆️

모든 방향에서 발언 가능

"당신의 직급이 높을수록, 다른 사람의 발언을 보호할 책임이 크다"

리더는 '말하는 사람'이 아니라 '들을 수 있게 만드는 사람'

🎯 리더 행동 3원칙

1. 마지막에 말하기

회의에서 리더가 먼저 의견을 내면, 나머지는 동조한다. → 리더는 마지막에 말한다.

2. 반대에 감사하기

반대 의견이 나올 때: "좋은 지적입니다. 더 얘기해 주세요" → 안전 신호 발송

3. 실수 먼저 공개하기

"지난주 내 판단이 틀렸다" → 실수 공개가 안전한 환경 조성
